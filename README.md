## 🛡️ "Shadow Knight"

Shadow Knight é um jogo de ação 2D desenvolvido em **Python** utilizando a biblioteca **Pygame**. O jogador deve sobreviver a ondas de inimigos, completar duas fases e alcançar a pontuação necessária para vencer.

O projeto foi desenvolvido com foco em **Programação Orientada a Objetos (POO)**, organização de códigos e boas práticas de desenvolvimento.

---

# 🎮 Funcionalidades

- ✅ Modo Single Player
- ✅ Modo Cooperativo (2 jogadores)
- ✅ Modo Versus
- ✅ Duas fases com dificuldade progressiva
- ✅ Sistema de spawn aleatório de inimigos
- ✅ Sistema de colisão entre entidades
- ✅ Sistema de tiros
- ✅ Tela de Game Over
- ✅ Sistema de pontuação
- ✅ Ranking Top 10 utilizando SQLite
- ✅ Música para Menu, Fases e Score

# 🎮 Controles

## Player 1

| Ação | Tecla |
|-------|--------|
| Mover para cima | ↑ |
| Mover para baixo | ↓ |
| Mover para esquerda | ← |
| Mover para direita | → |
| Atirar | Espaço |

---

## Player 2

| Ação | Tecla |
|-------|--------|
| Mover para cima | W |
| Mover para baixo | S |
| Mover para esquerda | A |
| Mover para direita | D |
| Atirar | CTRL Esquerdo |
